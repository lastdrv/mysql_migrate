Вспомогательный проект миграции с MySql на MariaDB


Шаги:
1. экспорт таблиц в sql дампы (export.py)
2. делаем create database <name> в новой базе (руками)
3. заливка таблиц в новую базу (import.py)
4. проверка что таблицы полностью залились (diff.py)
5. копируем пользователей (create_users.sh)
6. копируем права (grant_users.sh)
7. (не обязательно) архивирование и бакап дампов из п.1 (руками)

В окружении нужно иметь:
MYSQL_HOST_OLD = старый IP
MYSQL_HOST_NEW = новый IP
MYSQL_USER = пользователь (считаем что одинаковый на обоих базах)
MYSQL_PASS_OLD = старый пароль
MYSQL_PASS_NEW = новый пароль

Замечания:
1. хранимки и вьюшки переносим сами, тут такого нет
